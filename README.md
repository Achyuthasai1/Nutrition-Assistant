# 🥗 Nutrition Assistant

### Smart Nutrition & Wellness Management Platform

A full-stack MERN application that helps users track nutrition, monitor health progress, receive personalized diet plans, and connect with professional dietitians through a secure role-based system.

## 🚀 Quick Start

### Clone Repository

```bash
git clone <repository-url>
cd Nutrition-Assistant
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

## 🎯 User Roles

### 👤 User

* Track meals and nutrition
* Monitor BMI and health progress
* View personalized meal plans

### 👨‍⚕️ Dietitian

* Manage assigned clients
* Create and update meal plans
* Monitor client progress

### 🛡️ Admin

* Manage users and dietitians
* View platform analytics
* Monitor system activity
