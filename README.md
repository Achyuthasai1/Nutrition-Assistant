# 🥗 NutriSync AI

### Smart Nutrition & Wellness Management Platform

A full-stack MERN application that helps users track nutrition, monitor health progress, receive personalized diet plans, and connect with professional dietitians through a secure role-based system.

## ✨ Features

- 🔐 Secure JWT Authentication
- 👤 User Nutrition Dashboard
- 🥗 Personalized Meal Plans
- 📈 BMI & Progress Tracking
- 🤖 AI Nutrition Recommendations
- 👨‍⚕️ Dietitian Management Panel
- 🛡️ Admin Analytics Dashboard
- 📄 PDF & CSV Export
- 📱 Responsive Modern UI
